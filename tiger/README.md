# Tiger Docs (Docusaurus)

This directory is a Docusaurus-ready documentation site for BlueARC public release.

## Quick Start

```bash
cd tiger
npm install
npm run start
```

## Build

```bash
npm run build
```

## Structure

- `docs/whitepaper/` - new public whitepaper series
- `docs/admin/` - admin handover and operations
- `docs/legacy/` - existing sanitized project docs copied from root `docs/`

## Notes

- All personal OS paths were sanitized to `<repo-root>` placeholders.
- Add branding assets under `static/img/` if desired.
